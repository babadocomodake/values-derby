# 公式お題カード アーカイブ（原本）

実物のお題カードを**そのまま書き写した原本台帳**。ここが公式お題の出どころ（source of truth）。
ここから `src/topics.ts` の `OFFICIAL_TOPICS` へ変換する。評価は `docs/topic-design.md` の5条件に基づく。

## 運用ルール
- カードが届くたびに、下のテンプレを使って**1枚＝1セクション**で追記する。
- 選択肢はカードの **A〜G の順**で正確に書き写す（順番も原本どおり）。
- 評価は5条件（投影性 / 葛藤 / 会話喚起 / 軽さ・安全 / 割れる）を ◎○△ で。
- `採用` 欄に topics.ts のID（例 off-13）を書いたら変換済み。

### テンプレ
```
## カード#NNN
- 問い:
- A:
- B:
- C:
- D:
- E:
- F:
- G:
- ジャンル / トーン:
- 評価: 投影◯ 葛藤◯ 会話◯ 軽さ◯ 割れ◯
- メモ:
- 採用: （topics.ts のID / 未変換なら「未」）
```
